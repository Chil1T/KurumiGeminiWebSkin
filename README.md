# KurumiGeminiWebSkin
用于 Gemini Web 的《约会大作战》时崎狂三主题 CSS，可通过 [Stylus](https://add0n.com/stylus.html) 注入。

**安装方式：**
- **Stylus 用户：** 复制 [dist/Kurumiskin.user.css](dist/Kurumiskin.user.css) 的全部代码，在 Stylus 中为 gemini.google.com 新建样式并粘贴。  
- **或者直接安装：** [点击安装](https://raw.githubusercontent.com/Chil1T/KurumiGeminiWebSkin/main/dist/Kurumiskin.user.css)（需 Stylus 已安装）。

请开启深色模式使用。
